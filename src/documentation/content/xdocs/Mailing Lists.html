Qpid User List

The user's list is for discussions that relate to use or questions on Qpid. If you have questions about how a feature works, suggestions on additional requirements, or general questions about Qpid please use this list

  • To subscribe to the user's list send an empty e-mail to qpid-users-subscribe@incubator.apache.org
  • To remove yourself from the user's list send an empty e-mail to qpid-users-unsubscribe@incubator.apache.org
  • The user's mailing list is archived. You can view the archive at http://mail-archives.apache.org/mod_mbox/incubator-qpid-users/

Qpid Developer List

The developer's list is for discussions that relate to the on going development of Qpid. If you have questions about how a feature is being developed, suggestions on how to implement a new feature, or requests for a new feature this is the list to use.

  • To subscribe to the developer's list send an empty e-mail to qpid-dev-subscribe@incubator.apache.org
  • To remove yourself from the developer's list send an empty e-mail to qpid-dev-unsubscribe@incubator.apache.org
  • The developer's mailing list is archived. You can view the archive at http://mail-archives.apache.org/mod_mbox/incubator-qpid-dev/

Qpid Commits List

The commits list is for recieving notifications about code being committed to the Qpid repository. The trafic on this list is automatically generated by Subversion. You should not post messages to this list.

  • To subscribe to the commits list send an empty e-mail to qpid-commits-subscribe@incubator.apache.org
  • To remove yourself from the commits list send an empty e-mail to qpid-commits-unsubscribe@incubator.apache.org
  • The commits mailing list is archived. You can view the archive at http://mail-archives.apache.org/mod_mbox/incubator-qpid-commits/
